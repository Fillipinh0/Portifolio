Projetos em Destaque


Espaço dedicado aos projetos que estou desenvolvendo, com foco em criar aplicações eficientes e intuitivas. Aqui você encontrará uma prévia dos sistemas em andamento, onde aplico meus conhecimentos em desenvolvimento web e soluções escaláveis. Cada projeto representa um novo desafio e uma oportunidade de aprendizado contínuo.

  • Sistem de gestão de estoque

    Este sistema foi desenvolvido para a empresa Delícias Resendes com o objetivo de facilitar o controle de estoque e o acompanhamento dos produtos utilizados. Ele permite armazenar todos os produtos usados mensalmente, exibir o estoque atual, e oferece funcionalidades completas, como adicionar novos produtos, marcar produtos como usados, atualizar quantidades e deletar itens conforme necessário. Esse controle centralizado otimiza a gestão de insumos, promovendo uma visão clara do consumo e das necessidades de reposição.

  • SSFit

    O SSFit é um sistema completo de vendas de produtos fitness, que desenvolvi para oferecer uma experiência visualmente agradável e prática. O site possui todas as funcionalidades essenciais, desde a exibição de produtos até o processo de compra, com uma interface intuitiva e design pensado para facilitar a navegação e atender às necessidades dos clientes.
